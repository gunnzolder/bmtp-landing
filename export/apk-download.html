Welcome to the BMTP test prototype page!
To join the test you need:

  1. Mobile phone with Android
  2. Enable the installation of downloaded applications in the security settings (i.e. Allow installation of non-official apps)
  3. Download APK file
  4. Install the application
  5. If the downloaded file does not open right away, locate it in your Downloads folder using your file manager and open from there.
